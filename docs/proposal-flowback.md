# 글로벌 커머스 환불·보상·정산 통합 월렛

> 가칭: **FlowBack / ReFlow / StableFlow**

## 한 줄 정의

해외 구매자가 한국 셀러에게 상품을 구매한 뒤 발생하는 부분 환불, 반품, 보상, 포인트, 재구매 혜택을 하나의 스테이블코인 기반 월렛에서 처리하는 글로벌 커머스 정산 플랫폼.

## 프로젝트 배경 (Why)

단순히 "스테이블코인으로 결제합니다"라는 메시지는 약하다. 그러나 해외 직구·역직구·글로벌 커머스에서는 결제 이후 발생하는 문제가 많다.

- 배송 지연
- 일부 상품 품절
- 부분 환불
- 반품 비용 부담
- 셀러와 구매자 간 책임 소재
- 쿠폰·리워드·보상금이 따로 흩어짐
- PG/카드 환불은 며칠 걸림
- 해외 구매자는 "돈이 언제 돌아오는지" 불안함

이 프로젝트에서 스테이블코인은 결제수단이 아니라, **거래 이후 가치 흐름을 끊기지 않게 연결하는 수단**으로 사용된다.

---

## 신청서 본문

### 1. 한 줄 정의

글로벌 커머스 거래 이후 발생하는 환불·보상·리워드·재구매 혜택을 하나의 월렛 안에서 연결해, 구매자와 셀러 모두에게 예측 가능한 정산 경험을 제공하는 스테이블코인 기반 사후 정산 플랫폼입니다.

### 2. 문제 정의

해외 커머스 거래에서는 결제 이후 배송 지연, 부분 품절, 반품, 교환, 보상 지급 등 다양한 후속 정산 상황이 발생합니다. 그러나 현재 구조에서는 결제, 환불, 쿠폰, 포인트, 재구매 혜택이 각각 분리되어 있어 사용자는 환불 상태를 즉시 확인하기 어렵고, 셀러는 자금 회수와 고객 보상 운영을 별도로 관리해야 합니다. 특히 소규모 글로벌 셀러일수록 환불 지연과 보상 처리의 불투명성으로 인해 고객 신뢰와 재구매율이 낮아지는 문제가 발생합니다.

### 3. 핵심 솔루션

본 프로젝트는 구매 이후 발생하는 환불·보상·리워드·재구매 혜택을 하나의 스테이블코인 월렛에서 관리합니다. 주문별 정산 상태를 기록하고, 환불 예정 금액·보상 리워드·사용 가능 혜택을 사용자가 한 화면에서 확인할 수 있도록 설계합니다. 이를 통해 결제 이후 분리되어 있던 가치 흐름을 연결하고, 사용자가 다음 구매로 자연스럽게 이어질 수 있는 경험을 제공합니다.

### 4. 주요 기능

| 기능 | 설명 |
|------|------|
| 주문별 정산 상태 확인 | 결제 완료, 배송 중, 부분 환불 예정, 보상 지급, 재구매 가능 상태를 한 화면에서 확인 |
| 부분 환불 및 보상 지급 | 배송 지연, 품절, 교환 불가 등 상황에 따라 일부 금액을 스테이블코인 또는 리워드 형태로 지급 |
| 재구매 연계 월렛 | 환불금과 보상 리워드를 다음 구매에 바로 사용할 수 있도록 연결 |
| 셀러 정산 대시보드 | 환불 예정액, 보상 지급액, 고객 재구매 전환율을 관리 |
| 거래 상태 기록 | 주문별 정산·환불·보상 이력을 투명하게 기록해 분쟁 가능성을 축소 |

### 5. 타깃 사용자/시장

- **1차 타깃**: 해외 소비자를 대상으로 판매하는 K-뷰티, 패션, 굿즈, 디지털 상품 기반의 중소형 커머스 셀러. 글로벌 판매 과정에서 부분 환불, 배송 이슈, 고객 보상 처리 빈도가 높지만 자체 정산 인프라를 갖추기 어려운 사업자.
- **2차 타깃**: 한국 상품을 구매하는 해외 소비자. 환불 상태와 보상 혜택을 명확히 확인하고 다음 구매에 활용하고자 하는 사용자.

### 6. 비즈니스 모델

셀러 대상 SaaS 구독료와 거래 기반 수수료를 결합한 모델입니다.

- 예시 요금제: **월 구독료 3만 원 + 거래액의 0.3% 수수료**
- 장기적으로는 글로벌 커머스 플랫폼, 물류사, 크리에이터 커머스 운영사와 제휴하여 정산 인프라 형태로 확장.

### 7. 기존 솔루션의 한계

기존 결제·PG 시스템은 결제와 환불 처리에는 강점이 있지만, 거래 이후 발생하는 보상, 리워드, 재구매 유도까지 하나의 흐름으로 연결하지 못합니다. 카드 환불은 처리 시간이 길고, 쿠폰이나 포인트는 플랫폼 내부에 갇혀 있어 사용자가 실제 가치로 체감하기 어렵습니다. 또한 소규모 셀러는 환불·보상·재구매 데이터를 통합적으로 관리하기 어렵습니다.

### 8. 차별점

본 프로젝트는 스테이블코인을 단순 결제수단으로 사용하는 것이 아니라, 거래 이후 발생하는 가치 흐름을 연결하는 정산 인프라로 활용합니다. 환불, 보상, 리워드, 재구매 혜택을 하나의 월렛 경험으로 통합함으로써:

- **구매자**: 돈이 어디에 있고 언제 사용할 수 있는지 명확히 인지 가능
- **셀러**: 고객 보상과 재구매 전환을 하나의 운영 구조 안에서 관리 가능

### 9. XRPL을 선택한 이유

XRPL은 빠른 결제 확정성과 낮은 수수료 구조를 갖추고 있어 소액 환불, 보상, 리워드 지급처럼 빈번하고 작은 단위의 거래에 적합합니다. 또한 **Escrow 기능**을 활용하면 주문 상태에 따라 자금을 잠금·해제하는 구조를 설계할 수 있어, 구매 이후 환불·보상·정산 흐름을 자동화하기에 적합합니다. 이를 통해 글로벌 커머스에서 발생하는 거래 후 정산 문제를 낮은 비용과 빠른 속도로 처리할 수 있습니다.

## 핵심 가치 요약

| 항목 | 내용 |
|------|------|
| **핵심 컨셉** | 스테이블코인을 "결제수단"이 아닌 "사후 정산 인프라"로 활용 |
| **연결 대상** | 환불 + 보상 + 리워드 + 재구매 혜택 |
| **타깃 통화 흐름** | 한국 셀러 ↔ 해외 구매자 (역직구/직구 양방향) |
| **수익 모델** | SaaS 구독 + 거래 수수료 (예: 월 3만 원 + 0.3%) |
| **기술 기반** | XRPL (빠른 확정성, 낮은 수수료, Escrow) |
