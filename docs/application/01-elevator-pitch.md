# 01 · Elevator Pitch (한 줄 정의)

> **FlowBack은 K-커머스 글로벌 셀러와 해외 구매자를 위해, 결제 이후 발생하는 부분 환불·반품·보상·재구매 혜택의 분절된 가치 흐름을 XRPL Escrow와 RLUSD 스테이블코인 월렛으로 연결해 D+0 실시간 정산을 제공하는 사후 정산 인프라입니다.**

## 분해 (신청서 양식 매핑)

| 슬롯 | 내용 |
|------|------|
| 서비스명 | FlowBack |
| 타깃 사용자 | K-커머스 글로벌 셀러(K-뷰티/패션/굿즈/디지털 상품) + 해외 구매자 |
| 핵심 문제 (Pain) | 결제 이후 환불·보상·리워드가 분절·지연되어 신뢰·재구매율 하락 |
| 핵심 솔루션 방식 | XRPL native Escrow로 결제 자금을 락업하고, RLUSD 스테이블코인 월렛으로 환불·보상·재구매 혜택을 한 흐름으로 처리 |
| 서비스 카테고리 | 글로벌 커머스 사후 정산(Post-purchase Settlement) 인프라 |

## 한 문장 변형 (채널별)

- **투자자 대상 (60자)**
  글로벌 커머스의 환불·보상을 XRPL 위에서 묶어 D+2 → 5초로 단축하는 정산 인프라.
- **셀러 대상 (80자)**
  품절·지연이 발생해도 한 화면에서 부분환불·보상·재구매 혜택을 처리해, 고객을 다시 돌아오게 만드는 글로벌 커머스 정산 솔루션.
- **구매자 대상 (80자)**
  해외 직구에서 돈이 어디에 있는지·언제 돌려받는지·어떻게 다시 쓸 수 있는지 한 월렛에서 확인하는 글로벌 커머스 월렛.
