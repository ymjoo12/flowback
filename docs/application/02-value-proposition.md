# 02 · 핵심 가치 제안 / 차별화 포인트

기존 결제·PG·선정산·리워드 솔루션 대비 **속도, 수수료, 투명성, 자금 통합성** 4개 축에서 정량 개선을 제공합니다.

## 정량 비교

| 지표 | 카드/PG 환불 | 선정산 핀테크 | 플랫폼 포인트·쿠폰 | **FlowBack** |
|------|-------------|-------------|-----------------|-----------|
| 환불 도달 시간 | 영업일 기준 **3~7일** | **즉시(0~1일)**, 단 별도 신청 | N/A (포인트만 적립) | **3~5초** (XRPL ledger close) |
| 환불 처리 수수료(셀러 기준) | 약 **1.0~3.5%** (카드사·PG 합산) | **2~3%** | 0% (가치는 플랫폼에 락인) | **< $0.0001 / tx** (XRPL 네트워크 수수료) |
| 부분 환불 단위 | 카드 매출의 백분율 단위 | 거래 단위 | 정책에 따라 다름 | **임의 단위 (XRP drops/RLUSD 소수점)** |

> 카드 환불 처리 시간·수수료 추정치는 한국은행 「2024년 지급결제 동향」, 카드사·PG 공시 기준입니다. 선정산 수수료는 국내 선정산 핀테크 3사 공개 단가(2024-2025) 기준 평균입니다.

## 정성 차별점

1. **결제수단이 아니라 정산 인프라**
   스테이블코인을 "결제하는 방식"이 아닌 **"환불·보상·재구매를 묶어 처리하는 수단"**으로 사용한다. 동일 컨셉의 경쟁자가 부재함.

2. **별도 스마트컨트랙트 없이 ledger-level Escrow**
   XRPL의 native `EscrowCreate / EscrowFinish / EscrowCancel`로 결제 자금을 락업하므로, Solidity 컨트랙트 감사·취약점 리스크가 없다.

3. **셀러 운영 화면이 하나**
   환불·보상·재구매 전환율을 한 대시보드에서 본다. 기존에는 PG 콘솔(환불) + 마케팅 툴(쿠폰) + CRM(리워드)이 분리되어 있다.

4. **공개 검증 가능한 거래 로그**
   분쟁 발생 시 `testnet.xrpl.org` / `livenet.xrpl.org`에서 동일 해시로 정산 이력을 확인할 수 있다. PG 환불 내역은 사후 검증이 어렵다.

## 한 줄 요약 (신청서 양식)

> 기존 카드 환불 대비 **환불 도달 시간 3~7일 → 3~5초**, 처리 수수료 **1.0~3.5% → < $0.0001/tx**, 그리고 환불·보상·재구매 데이터를 **분리된 3개 시스템 → 1개 월렛**으로 통합.
